# Deskkit - Downloads

The all-in-one desktop toolkit for Mac, Windows, Linux, and ChromeOS.

## Download the latest version

Grab the file for your system from the [Releases page](../../releases/latest):

- **Windows** - `Deskkit-Windows.exe` (double-click to run)
- **Mac** - `Deskkit-macOS.zip` (unzip, then right-click the app and choose Open the first time)
- **Linux / ChromeOS** - `Deskkit-Linux` (`chmod +x Deskkit-Linux`, then run)

Deskkit checks here automatically and offers a one-click download when a new version is out.

_This repository hosts the app downloads only._
